RARA Avatar Tools

アバター軽量化・Quest/iOS対応のエディタ拡張ツール群

VCC / ALCOM に追加

上のボタンを押すと VCC (VRChat Creator Companion) または ALCOM が起動し、リポジトリの追加確認画面が開きます。

手動で追加する場合

VCC / ALCOM の Settings > Packages > Add Repository に次の URL を貼り付けてください。

https://rara-vrc.github.io/rara-avatar-tools/index.json

収録ツール

  • Quest Converter — Quest/iOS 向けのマテリアル・メッシュ変換と軽量化
  • PC Optimizer — PC 向けアバターの最適化
  • Avatar Studio — 変換・診断・アップロード補助の統合ウィンドウ

ALCOM (anatawa12 版の代替クライアント) でも、同じボタン・同じ URL で追加できます。
