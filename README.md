# RARA Avatar Tools (VPM リポジトリ)

VRChat アバターの軽量化と Quest/iOS 対応を支援する Unity エディタ拡張パッケージ `jp.rara.avatar-tools` を、VCC / ALCOM で追加できる VPM リポジトリとして配信するためのリポジトリです。収録ツールは Quest Converter・PC Optimizer・Avatar Studio の 3 種です。

このリポジトリはリリース／リスティング配信専用です。GitHub Actions がリリース zip と VPM リスティング (`index.json`) を生成し、GitHub Pages で配信します。

## 導入 (VCC / ALCOM)

- ワンクリック追加: 追加ページ https://rara-vrc.github.io/rara-avatar-tools/ を開き、「VCC / ALCOM に追加」ボタンを押します。
- 手動追加: VCC / ALCOM の Settings > Packages > Add Repository に次の URL を貼り付けます。
  - https://rara-vrc.github.io/rara-avatar-tools/index.json

追加後、対象のアバタープロジェクトの Manage Project から `RARA Avatar Tools` を導入できます。
