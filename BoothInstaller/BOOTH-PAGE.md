# RARA Avatar Tools(VCC / ALCOM 導入)

> **Open β(オープンベータ)公開中**
> VRChat アバター向けの便利ツール詰め合わせです。現在オープンベータとして公開しており、無償でお試しいただけます。

---

## 概要

RARA Avatar Tools は、VRChat のアバター改変・セットアップを補助する Unity 向けツール群です。
本商品(unitypackage)は **導入案内(ガイド)のみ** を含みます。ツール本体は含まれていません。
本体は **VCC(VRChat Creator Companion)/ ALCOM** から導入する仕組みのため、常に最新版を安全に利用できます。

- ワンクリックで VCC / ALCOM にリポジトリを追加する導入ガイド付き
- 本体はリポジトリ配信のため、アップデートが簡単
- Unity 2019.4 以降で動作(案内ガイド自体は依存ゼロ)

---

## 対応環境

- **Unity**: 2019.4 以降(VRChat SDK 対応バージョン推奨)
- **導入管理**: VCC(VRChat Creator Companion)または ALCOM
- **プラットフォーム**: Windows / macOS(VCC・ALCOM が動作する環境)

---

## 導入方法(VCC・ALCOM)

> **重要:** この unitypackage は「案内のみ」です。ツール本体はこの中には入っていません。
> 下記の手順で VCC / ALCOM からリポジトリを追加して導入してください。

1. 本商品の `RARA-AvatarTools-VCC-Guide.unitypackage` を Unity プロジェクトにインポート
   - インポート後、自動で導入ガイドウィンドウが開きます
   - 手動で開く場合はメニュー **「RARA > 導入ガイド(VCC・ALCOM)」**
2. ガイド内の **「VCC / ALCOM にリポジトリを追加」** ボタンを押す
   - VCC / ALCOM が起動し、リポジトリ追加の確認が表示されます
   - ボタンが反応しない場合は、下記 URL を VCC / ALCOM に手動で追加してください
     ```
     https://rara-vrc.github.io/rara-avatar-tools/index.json
     ```
3. プロジェクトに **「RARA Avatar Tools」** を追加
4. Unity を開くと、メニューバーに **「RARA」** が表示されます

導入ページ: https://rara-vrc.github.io/rara-avatar-tools/
